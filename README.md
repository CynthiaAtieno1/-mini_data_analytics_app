# -mini_data_analytics_app
# Mini Data Analytics App (BSD 314)

This project analyzes customer sales data using Python. It reads a CSV file, cleans the data, performs some analysis, and shows visual charts.

## Setup
1. Create a virtual environment  
   `python -m venv venv`
2. Activate it  
   - Windows: `venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`
3. Install requirements  
   `pip install -r requirements.txt`
4. Run the project  
   `python main.py`

## Output
- CSV file of categorized customers
- Bar chart (sales by category)
- Histogram (purchase distribution)

## What I Learned
I learned how to use pandas for data cleaning, matplotlib for visualization, and how to organize Python code into modules.
